# Cybersecurity Concepts & Tools Glossary

## Network Security

### Tools & Technologies
- **Wireshark**: Network protocol analyzer for packet capture and analysis
- **tcpdump**: Command-line packet analyzer
- **Nmap**: Network mapping and port scanning tool
- **NetCat**: Networking utility for reading/writing network connections

### Concepts
- **OSI Model**: Seven-layer model for network communications
- **TCP/IP**: Core protocols for internet communications
- **Ports & Protocols**: Standard network service port numbers and protocols
- **Firewall**: Network security system that monitors and controls traffic

## Cryptography

### Tools
- **OpenSSL**: Cryptography toolkit for SSL/TLS
- **GPG**: Implementation of PGP for encryption
- **SSH**: Secure Shell protocol for encrypted communications

### Concepts
- **Symmetric Encryption**: Same key for encryption/decryption
- **Asymmetric Encryption**: Public/private key pairs
- **Hashing**: One-way functions for data integrity
- **Digital Signatures**: Authentication and non-repudiation

## SIEM & Log Analysis

### Tools
- **Splunk**: Platform for searching, monitoring, and analyzing machine data
- **ELK Stack**: Elasticsearch, Logstash, and Kibana for log analysis
- **Nagios**: System and network monitoring

### Concepts
- **Log Aggregation**: Centralizing log data
- **Event Correlation**: Linking related security events
- **Alerting**: Automated notification of security incidents
- **Baselining**: Establishing normal behavior patterns

## Vulnerability Assessment

### Tools
- **Nessus**: Vulnerability scanner
- **Metasploit**: Penetration testing framework
- **OWASP ZAP**: Web application security scanner

### Concepts
- **CVE**: Common Vulnerabilities and Exposures
- **Risk Assessment**: Evaluating security risks
- **Patch Management**: Updating systems for security
- **Penetration Testing**: Authorized system testing

## Incident Response

### Tools
- **FTK**: Forensic Toolkit for digital investigation
- **Volatility**: Memory forensics framework
- **Autopsy**: Digital forensics platform

### Concepts
- **Chain of Custody**: Documentation of evidence handling
- **Incident Handling**: Structured approach to security incidents
- **Memory Analysis**: Examining system RAM
- **Digital Forensics**: Scientific examination of digital evidence

## Access Control

### Tools
- **Active Directory**: Directory service for Windows networks
- **LDAP**: Lightweight Directory Access Protocol
- **OAuth**: Open standard for access delegation

### Concepts
- **Authentication**: Verifying identity
- **Authorization**: Granting access rights
- **Role-Based Access**: Access based on user roles
- **Multi-Factor Authentication**: Multiple verification methods

## Cloud Security

### Tools
- **AWS CloudWatch**: Monitoring for AWS resources
- **Azure Security Center**: Security management for Azure
- **Cloud Trail**: AWS API activity logging

### Concepts
- **Shared Responsibility**: Security model for cloud services
- **Cloud Access Security**: Managing cloud service access
- **Container Security**: Securing containerized applications
- **Cloud Compliance**: Meeting regulatory requirements

## Web Security

### Tools
- **Burp Suite**: Web vulnerability testing
- **ModSecurity**: Web application firewall
- **SSL Labs**: SSL/TLS testing tools

### Concepts
- **XSS**: Cross-site scripting attacks
- **SQL Injection**: Database attack technique
- **CSRF**: Cross-site request forgery
- **Web Application Firewalls**: Application-level protection

## Best Practices & Standards

### Frameworks
- **NIST**: National Institute of Standards and Technology
- **ISO 27001**: Information security standard
- **CIS Controls**: Critical security controls

### Concepts
- **Defense in Depth**: Layered security approach
- **Least Privilege**: Minimal access rights
- **Security Awareness**: User education and training
- **Compliance**: Meeting regulatory requirements
